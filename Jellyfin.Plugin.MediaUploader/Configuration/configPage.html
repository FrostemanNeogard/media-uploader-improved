Upload Page

Go to Upload Page

Click on the "Go to Upload Page" button (or navigate to ) to visit the page where you upload media. This page allows non-admin users to upload files using their personal API Key.

You may want to bookmark this link for easy access.

How to use the Upload Page:

  1. Generate an API Key: Each user needs their own API Key. Go to your Jellyfin user profile (click profile icon top right) → Settings → API Keys → Click the '+' icon to create a new key. Give it a descriptive name (e.g., "MediaUploader Key") and copy the generated key value.
  2. Go to the Upload Page: Click the "Go to Upload Page" button or open the link above. Paste your generated API Key into the designated field. Select the media file(s) (or alternatively a directory containing media files) that you want to upload. Specify where you want to save them using the "Destination" field, and optionally the "Subfolder" field.
  3. Upload your files: Click the "Upload" button to begin your upload(s). If you selected multiple files or a directory, your files will upload one by one with a progress bar for each file.
  4. Upload your files: Click the "Upload" button to begin your upload(s). If you selected multiple files or a directory, your files will upload one by one with a progress bar for each file. Note that if you upload a directory, any non-video files and any files in subdirectories will be ignored.

Congratulations! Your new media is now accessible on your server.
